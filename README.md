## How to Play

1. Make sure you have Python installed on your system
2. Clone or download this repository
3. Run the game:
```bash
   python rock_paper_scissors.py
```
4. Choose your move:
   - Type `0` for Rock
   - Type `1` for Paper
   - Type `2` for Scissors
   - Type `Q` to quit and see final score
